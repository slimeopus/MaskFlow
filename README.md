# MaskFlow — Blur Personal Data

[![Chrome Web Store](https://img.shields.io/badge/Chrome_Web_Store-Extension-green?logo=google-chrome)](https://chrome.google.com/webstore)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

**MaskFlow** — расширение для браузера, которое размывает конфиденциальные данные на веб-страницах: балансы карт, email, номера телефонов, переписку и аватары. Идеально для безопасных скриншотов и демонстраций экрана.

## ✨ Возможности

- **Балансы и деньги** — автоматическое размытие сумм, номеров карт, счетов
- **Номера телефонов** — маскировка всех форматов телефонных номеров
- **Email-адреса** — защита от утечки email-контактов
- **Переписка** — размытие сообщений в мессенджерах, чатах, комментариях
- **Аватары и фото** — маскировка профильных изображений и лиц
- **Показывать по наведению** — временный раскрытие данных курсором
- **Ручное размытие** — кисть для выборочного скрытия любых элементов
- **Автоматизация** — настройка автоматического размытия для конкретных сайтов

## 🛠 Установка

### Из Chrome Web Store (рекомендуется)

[Установить из Chrome Web Store](https://chrome.google.com/webstore)

### Ручная установка

1. Скачайте архив с расширением
2. Откройте `chrome://extensions/` в браузере Chrome
3. Включите **Режим разработчика** (Developer mode)
4. Нажмите **Распаковать расширение** (Load unpacked)
5. Выберите папку с расширением

## 📖 Как использовать

1. **Включите расширение** — кликните на иконку MaskFlow в панели инструментов
2. **Выберите категории** — отключите/включите категории данных, которые нужно скрывать
3. **Сделайте скриншот** — данные будут размыты на странице
4. **Используйте умные режимы**:
   - **Hover to Reveal** — наведите курсор на размытый элемент, чтобы временно его увидеть
   - **Blur Brush** — включите кисть и кликните на любой элемент для размытия

## ⚙️ Архитектура

### Файловая структура

```
MaskFlow/
├── manifest.json          # Конфигурация расширения (Manifest V3)
├── background.js          # Service worker для управления состоянием
├── content/
│   ├── content.js         # Основная логика размытия данных
│   └── blur.css           # CSS-стили для размытия
├── popup/
│   ├── popup.html         # Интерфейс всплывающего окна
│   ├── popup.js           # Логика управления расширением
│   └── popup.css          # Стили интерфейса
├── icons/                 # Иконки расширения
└── README.md              # Документация
```

### Технологии

- **Manifest V3** — современный API Chrome Extensions
- **MutationObserver** — отслеживание изменений DOM
- **Regular Expressions** — распознавание паттернов данных
- **Chrome Storage API** — сохранение настроек пользователя
- **CSS Filters** — размытие изображений и видео

### Паттерны распознавания

- **Балансы**: `$123`, `1000 ₽`, `1234-5678-9012-3456`
- **Телефоны**: `+7 999 123-45-67`, `8 (999) 123-45-67`
- **Email**: `user@example.com`
- **Сообщения**: элементы с `role="chat"`, `data-testid*="message"`
- **Аватары**: изображения с классами `avatar|profile|photo|userpic`

## 🔒 Приватность

MaskFlow обрабатывает данные **локально в браузере**. Никакая информация не отправляется на внешние серверы. Все настройки хранятся в локальном хранилище Chrome.
